# Adidas Landing Page | Responsive Web Design

A modern, high-performance **Landing Page** for Adidas, built with a focus on clean UI/UX, responsiveness, and advanced CSS techniques. 
This project demonstrates the ability to translate brand identity into a functional, interactive web experience.

## Project Overview

This project showcases a sleek **Glassmorphism** design. 
The goal was to create a fluid user experience that maintains its visual integrity across all devices, from high-resolution monitors to mobile screens.

### Key Technical Features:

* **Glassmorphism UI:** Implemented using `RGBA` color values and `backdrop-filter` for a premium frosted-glass effect.
* **Dynamic Backgrounds:** Leveraged CSS `::after` pseudo-elements combined with `clip-path: circle()` to create unique, brand-aligned shapes.
* **Advanced Flexbox Layout:** Utilized Flexbox for precise alignment and distribution of elements within the container.
* **Responsive Media Queries:** Crafted custom breakpoints to ensure a seamless transition between desktop and mobile layouts.
* **Interactive Brand Elements:** Integrated `hover` states and transitions on navigation links to enhance user engagement.

## Tech Stack

* **HTML5:** Semantic markup for SEO and accessibility.
* **CSS3:** Custom styling, Flexbox, and Media Queries.
* **Google Fonts:** Typography integration.
